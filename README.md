# resource-zhcn-main

本仓库用于归档和发布多个独立 HTML 页面。这些页面内容多样，不针对任何特定域名或具体网站，仅供学习、参考和存档使用。

## 项目简介

这是一个资源归档仓库，主要存放各类独立的 HTML 文件。每个页面均为自包含的静态页面，不依赖外部动态接口或特定后端服务。仓库旨在保存一些有价值的页面素材，方便长期访问和回顾。

## 目录说明

- `pages/` — 存放所有归档的 HTML 页面文件
- `assets/` — 页面所使用的静态资源（图片、样式、脚本等）
- `index.html` — 仓库首页，提供页面列表和快速导航

## 页面归档说明

每个 HTML 页面均以独立文件形式存储，文件名清晰反映页面内容或用途。页面可能包含以下类型：

- 信息展示页
- 工具类页面
- 学习笔记或文档页
- 其他实用静态页面

所有页面均为历史存档版本，反映归档时的状态，后续可能不定期更新或补充。

## 使用方式

1. 直接打开 `index.html` 查看页面列表
2. 点击对应链接访问具体页面
3. 也可直接访问 `pages/` 目录下的 HTML 文件
